# WGAN-GP 项目说明

## 项目概述
本项目是 **2025/2026 秋季《人工智能原理与应用》课程** 的大作业，实现了 **Wasserstein GAN with Gradient Penalty (WGAN-GP)** 模型。

## 数据集信息
- **尺寸**：96 × 96 × 3（高度 × 宽度 × 通道数）  
- **注意事项**：若更换为其他数据集，需相应调整模型输入尺寸、预处理流程及相关超参数。

## 项目状态
- **当前版本**：v1.0
- **存在问题**：
  - 生成图片精准度不足  
  - 生成图片清晰度较低  
- **未来计划**：
  - 持续优化模型结构与训练策略  
  - 提升生成图片质量

## 项目结构
-项目根目录
-├─ dataset          # 数据集目录
-│  └─ faces         # 数据集（96×96×3）
-├─ src              # 源码目录
  - ├─ models            # 模型子目录
  - ├─ output            # 生成结果输出目录
  - ├─ discriminator.py  # 判别器网络实现
  - ├─ generator.py      # 生成器网络实现
  - └─ main.py           # 训练/测试主程序
